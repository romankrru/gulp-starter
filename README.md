## Как пользоваться

Для работы необходим [Node.js](https://nodejs.org/)

### Установка зависимостей:

```
npm i
```

### Запуск development сервера:

```
gulp
```
или
```
gulp develop
```

### Сборка проекта:

```
gulp build
```

### Запуск сервера для собранного проекта:

> Изменения в исходном коде не будут отображаться в браузере

```
gulp serve
```

### Стурктура проекта
- `.tmp`
- `build`
- `config`
- `src`
  - `blocks`
  - `common`
  - `fonts`
  - `img`
  - `pages`
    - `parts`
